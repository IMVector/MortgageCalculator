# 房贷计算器

一款专业的房贷计算工具，支持商业贷款、公积金贷款及组合贷款计算。

## 功能特点

- 📊 **多种贷款类型** - 支持商业贷款、公积金贷款、组合贷款
- 💰 **两种还款方式** - 等额本息、等额本金
- 📅 **详细还款计划** - 查看每期本金、利息、剩余本金
- ⚡ **提前还款模拟** - 支持缩短期限或减少月供
- 💾 **数据持久化** - 自动保存贷款方案
- 📤 **结果分享** - 导出还款计划

## 技术栈

- Swift 5.0
- SwiftUI
- MVVM 架构

## 系统要求

- iOS 17.0+
- Xcode 15.0+

## 安装

1. 克隆仓库
```bash
git clone https://github.com/your-username/MortgageCalculator.git
```

2. 打开 `MortgageCalculator.xcodeproj`

3. 选择模拟器或真机运行

## 使用说明

1. 在「贷款输入」页面添加贷款信息
2. 点击「保存并计算」查看还款计划
3. 在结果页可以添加提前还款模拟
4. 点击还款段展开查看每期详情
